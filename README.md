# windowManager
simple dynamic tiling window manager for Xorg

# Usage
1. Clone the repository
```
git clone https://github.com/kuglatec/windowManager
```
2. Install Xephyr for using a nested X session
3. ```cd windowManager```
4. ```./buiild_and_run.sh```
